# Windows Setup Configuration

## Decluttering Tools

### For Ease of Use (Graphical Interface)

* Winhance: A user-friendly tool that helps you remove pre-installed apps, disable features like Copilot and Bing search, and manage privacy settings.
* Windows 11 Debloater: A popular tool with a graphical interface that allows you to select what to remove. It offers different presets for desktops, laptops, and virtual machines.

### For Advanced Users (Command-line)

* Win11Debloat: A powerful, open-source PowerShell script for removing a wide range of pre-installed applications, disabling telemetry, and customizing the user interface.
